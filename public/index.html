Required Worker secrets

ANTHROPIC_API_KEY — required

ANTHROPIC_MODEL — optional (defaults to claude-sonnet-4-20250514)

ALLOWED_ORIGIN — optional (useful for split front-end / API origins)

Deploy steps

1. npm install

2. npx wrangler secret put ANTHROPIC_API_KEY

3. npx wrangler deploy

4. Attach rancher.everlightos.com as a custom domain in the Cloudflare dashboard.

Local development

Create a .dev.vars file with ANTHROPIC_API_KEY=sk-ant-... then run npm run dev (port 5000).
